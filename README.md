# CodeAlpha_Superstore-EDA-Project
## Project Overview
This project performs Exploratory Data Analysis (EDA) on a Superstore sales dataset.

## Objectives
- Understand dataset structure
- Detect missing values and duplicates
- Analyze sales and profit trends
- Explore category and regional performance
- Perform correlation analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Steps
1. Data Loading
2. Data Inspection
3. Data Cleaning
4. Univariate Analysis
5. Bivariate Analysis
6. Correlation Analysis
7. Business Insights

## Key Visualizations
- Sales by Category
- Profit by Region
- Monthly Sales Trend
- Correlation Heatmap

## Business Insights
- Best-performing categories can be identified.
- Regional profitability can be compared.
- Seasonal sales patterns can be observed.
- Relationships between Sales, Quantity, and Profit can be evaluated.
